Table

The table tag outlines a table in your web page or website.

The table tag consists of ;

  1. Table head.
  2. Table row.
  3. Table data.

The Table head are item on the top of the table.

The Table row are how many the table data to be represented are.

The Table data are elements to be represented in the table.
