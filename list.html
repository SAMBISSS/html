list

The list tag in html is used for creating list.

There are two types of listing in html;

The first one is the ORDERED LIST The ordered list is a type of list in which the listing appear in a arragned order.

Below is an example of the ordered list;

  1. NAME
  2. DEPARTMENT
  3. LEVEL
  4. DATE

Another type of list is the UNORDERED LIST

The unordered list is a type of list in which the listing appears in a bullet order. Below is an example of an UNORDERED list;
  • NAME
  • DEPARTMENT
  • LEVEL
  • DATE
